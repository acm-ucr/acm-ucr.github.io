What is jQuery?

  • jQuery is a fast, small, and feature-rich JavaScript library
  • The "write less, do more" javascript framework
  • Provides a javascript API that makes client-side javascript easier
  • Provides similar APIs for user interface design (jQueryUI), mobile compatibility (jQueryMobile), unit testing (QUnit), and a css selector enginge (Sizzle)

Why jQuery?

  • Lightweight (32kB minified), CSS3 compliant, Cross-Browser support
    • jQuery is used on half of all websites
  • Makes HTML document traversal and manipulation, event handling, and AJAX really easy
Compare the following examples: 
// Want to get text of element w/ ID = example
console.log(document.getElementById("example").innerHTML);
// Now again, with jQuery
console.log($("#example").text());

Review of CSS selectors w/ jQuery examples

  • jQuery allows you to query the HTML document using $(called the "Bling" operator) 
    // Outputs the HTML of any element with class
// content inside of the body tag
$("body .content").html();
  • Notice the "Bling" operator takes in CSS Selectors as arguments
  • This is one of the features of jQuery that makes it extremely powerful and easy to use

Basic Selectors w/ jQuery Syntax

  • Target a certain HTML element 
    $("button").css("text-decoration", "underline");
  • The most basic selectors 
    // These will return a list of elements

// Select all elements named "element"
$("element");

// Select all elements with an ID named "ID"
$("#ID");

// Select all elements with a class named "class"
$(".class");
  • Selectors can be comma-separated 
    // Applies to all buttons and headers
$("h1, h2, h3, h4, h5, h6, button")
    .css("background", "#2d6cc0")
    .css("color", "white");
  • Wildcard selection 
    // Selects everything
$("*").css("display", "none");

// Alternatively, `hide` does the same thing and you control the speed
$("*").hide("slow");
  • Contextual Selection 
    // Select all spans inside of divs
$("div span");

// Select all paragraphs whose parents are divs
$("div > p");

// Or, use `.children()` with a selector
$("div").children("p");

// Select all spans immediately after divs
$("div + span");

// Or, use `.after()` with a selector
$("div").after("span");
  • Selection by attribute 
    // Select every element with attribute "title"
$("[title]");

// Select every element with attribute "href" == "https://bit.ly"
$("[href=https://bit.ly]");

// Select every img with attribute "title" that has "hello"
$("img[title~=hello]");

Pseudo-selectors

  • Recall that with CSS we could select elements based on their state w/ pseudo-selectors 
    /* Selects a button when it gets hovered */
button:hover { background: #f1ab00; }
  • jQuery lets us leverage Javascript plus selectors to make it simpler and more powerful! 
    $("button").hover(function() {
    $(this).css("background", "#f1ab00");
    alert("You hovered!");
});

Adding interactivity to your page with jQuery

  • As you saw in the last example, you can add logic to an element's state w/ jQuery
  • This is a common pattern in javascript web development
  • For example, show/hide on click: 
    $("#showDemo button").click(function() {
// Toggle switches between show and hide.
$(this).prev().toggle("slow");
});
  • You can achieve different effects by calling .show() or .hide() separately, or by using .slideToggle()
Recall an example from the 4th workshop
  • In CSS, you can change the font of your text using the font-family rule: 
    #fontExample {
  font-family: Arial, sans-serif;
  font-family: Georgia, serif;
		      
  /* Don't really need the serif specification */
  font-family: Times New Roman;
}
  • By default, this only works with standard browser fonts
Well, here's the jQuery code for that: 
$("#fontCTRL button").click(function() {
   /* All the buttons have an ID that corresponds to their font family
     I replace spaces in the ID with underscores, so I replace those 
     back */
   id = $(this).attr("id").replace("_", " ");

   // Now to actually change the font-family
   $("#fontExample").css("font-family", id);
   $("#fontExample code").css("font-family", id);
});
  • .attr() is from jQuery and allows you to access the element's attributes
  • .css() is from jQuery and allows you access certain css properties. You can pass in one argument to access the value, or use two to change the value.
  • .replace() is from javascript and allows you to do a regex replace on a string

Loading content asychronously with javascript (AJAX)

  • Ever wonder how Facebook and other sites refresh the page without you pressing anything? That's AJAX
  • AJAX stands for Asychronous Javascript and XML, although people don't really use XML anymore for transferring data (they use JSON instead).
  • AJAX allows you to ask an outside page for information with javascript and use the results in whichever way you'd like. You can use a "GET" request to get data.
  • You can also use AJAX to submit data to another page without loading the page. You can use a "POST" request to send data.

A simple AJAX example

  • Say we wanted to replace the text in the box below with another website's content
    • Some random text
    $("#ajaxButton").click(function() {
   $.ajax({ url: "facebook.com" }).done(function(data) {
       $("#ajaxDemo").html(data);
   });
